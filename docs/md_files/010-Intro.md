# Welcome to Lacuna Notebook

## Start Here

The goal of the Lacuna Notebook is to provide information that will help players get the
most out of their playthrough.  X4 is big, complicated, and obfuscated by lack of
instruction.  

The information provided in the Notebook entries tries to fill the gaps of in-game
documentation but it will also include spoilers.  If the entry has spoiler information,
it will have a [Spolier] tag at the beginning of the title.  A best-effort will be made
to note if any entry has spoilers but sometimes it could be matter of opinion.  

The format of the entries is in Markdown style.  X4 has no support for rendering 
Markdown content, but all the entries in the Notebook come from Markdown documents.  
More information can be viewed in the entries:

- Viewing Entries Offline
- Customizing Entries

This notebook is my first mod and is basically a copy of the Encyclopedia with code
bolted on for Lacuna Notebook.  All the functionality and content of the vanilla
Encyclopedia is in Lacuna Notebook also.

There is so much content I want to add that the organization of it will likely change
as more entries get added.  I also want to make sure this mod doesn't break anything but
no refunds :)!  Expect lots, no, tons of typo, grammatical, and formatting errors to, no
wait, too I mean...

If wondering where to go next, here are a couple of categories to visit next:

- For new players, open up "New to X4"
- For returning players, open up "Returning to X4"

## Searching Information

- The search function works just like the one in the Encyclopedia
- Searches are done only on the title of an entry, not the body/contents of it

## Avoiding Spoilers

- A best-effort will be made to include the [Spoiler] tag in the title of any entry that
  may have a spoiler

## Known Issues with the Mod

- When Lacuna Notebook is opened, the Encyclopedia icon in the top bar displays as
  active instead

## Planned Features for Future Updates

- Addition of a back and forward button for navigating history like a browser
- Support for jump entries, those entries with book icons that go instantly to an entry
- Support for images and possibly videos in the entries

## Viewing Entries Offline

- All content is generated from Markdown files in the docs/md_files directory of the mod
- For example, in Windows, using the standard Steam X4 installation dir, the Markdown
  files would be located in
  - C:\Program Files (x86)\Steam\steamapps\common\X4 Foundations\extensions\lacuna_nb\docs\md_files
- Everythign is written to those Markdown files and then a script is used to convert it
  to the 0001.xml t-file used by X4
- Entries can also be customized, see "Customizing Entries in the Notebook" for info
