- Reimagined Teladi and Argon Capital Ships
- New sectors and landmarks
- Redesigned Xenon models PE and SE

### 7.50 Flight Model Update and Hyperion Pack DLC | Released 2025-02-20

- [Relase Post on Steam](https://store.steampowered.com/news/app/392160/view/546725769035907147)
- [Changelog](https://forum.egosoft.com/viewtopic.php?p=5267469#p5267469)

#### Notable Changes

- Complete overhaul of ship flight dynamics
- Boosting now has its own dedicated energy pool instead of using shield energy
- Ship Rebalancing: Every ship in the game has been rebalanced, with refined engine,
  thruster, and component interactions.
- New Feature Behavior Inspection Mode: A new tool to analyse and adjust ship
  behaviours, giving you the ability to diagnose and solve problems with the ships in
  your fleet
- Added Wide Sensor Array Module for Stations: A new radar station module enhances
  detection capabilities, expanding your strategic awareness

## X4 Updates Notes for Version 8.00 Releases

### 8.00 Diplomacy Update and Envoy Pack DLC | Released 2025-09-10

- [Relase Post on Steam](https://store.steampowered.com/news/app/392160/view/612045811444875943)
- [Changelog](https://forum.egosoft.com/viewtopic.php?p=5290377#p5290377)

#### Notable Changes

- New Feature: A Diplomacy system built around three main components:
  - Embassy Room: An embassy facility that can be unlocked at the Player Headquarters
    (PHQ)
  - Agents: A new class of hireable NPCs who can be specialised in negotiation or
    espionage
  - Diplomatic Interference: Diplomatic standing between NPC factions can now be
    triggered or influenced
- New Feature: DLSS Frame Generation

## X4 Updates Notes for Version 9.00 Releases

### 9.00 Empire Update | Released 2026-06-10

- [Relase Post on Steam](https://store.steampowered.com/news/app/392160/view/503982156237767751)
- [Changelog](https://forum.egosoft.com/viewtopic.php?p=5318116#p5318116)

#### Notable Changes

- Major rebalancing of weapons and missiles
  - Added dedicated missile launcher slots on some S/M ships
- New Feature Travel Drive Stability System: M size and larger ships no longer drop out
  of Travel Drive instantly after receiving any damage.  Instead, they are increasingly
  difficult based on size to knock out of Travel Drive.
  - [Dev Post on Egosoft Forum](https://forum.egosoft.com/viewtopic.php?t=474552)
- Mining Region Overhaul: Completely overhauled how resource areas are managed
  - Removed the influence of resource probes on mining efficiency; they now indicate to
    miners where to focus their efforts
- New Feature Priority Orders: New UI functionality that allows instantly responsive
  "RTS"-style orders to be issued
  - Orders can be issued and queued from the map using [SHIFT] + [Right Click], while
    Priority Orders can be sent with [CTRL] + [Right Click] and queued using
    [CTRL] + [SHIFT] + [Right Click]
- S and M Kha'ak wrecks can now be recycled into Allographyne which transmutes to
  replace any ware in the production process
- Engines and Shields are now integrated into S and M ship models
  - Most equipment is now interchangeable between S/M ships of all commonwealth races
- Added target elements for torpedoes
- Added Ship Showroom Module: Allows players to store and display their favourite S and
  M ships on stylish landing pads
- Adjustments to the UI's visual style in all areas of the game
- NPC factions can better utilize and expand upon predefined construction plans for
  better station designs
- New Feature: FSR Frame Generation
